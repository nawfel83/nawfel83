## 🚀 Projets mis en avant (Featured Projects)

### 🏨 HBnB — Airbnb-like Backend Application
> Une application backend complète inspirée d'Airbnb, développée dans le cadre du cursus Holberton School.
* **Techs :** `Python`, `Flask`, `Flask-RESTX`, `SQLAlchemy`, `MySQL`, `JWT`
* **Focus :** Architecture en couches, API REST, authentification, modélisation de données, persistance.

### 🐚 Simple Shell
> Un interpréteur de commandes UNIX minimaliste écrit de zéro en C.
* **Techs :** `C`, `Linux`, `System Calls`
* **Focus :** Création de processus, parsing de commandes, gestion de l'environnement, résolution du `PATH`, gestion de la mémoire.

### 🖨️ _printf
> Une réimplémentation personnalisée de la célèbre fonction standard `printf` en C.
* **Techs :** `C`, `Variadic Functions`
* **Focus :** Parsing de formats, programmation modulaire, gestion des flux de sortie bas niveau.

### 🌐 Web Back-End Projects
> Une collection de projets backend axés sur l'écosystème JavaScript moderne.
* **Techs :** `JavaScript`, `Node.js`, `Express.js`
* **Focus :** Organisation de serveurs, promesses, programmation asynchrone, routage d'API, fondamentaux du backend.

### 🎨 Web Front-End Projects
> Des projets d'intégration et d'interface basés sur les fondamentaux du Web.
* **Techs :** `HTML5`, `CSS3`, `JavaScript`
* **Focus :** Structure sémantique, design réactif (responsive), manipulation du DOM, intégration d'interfaces utilisateur.
